About

What is Tank Hitter

Tank Hitter is a game client application created specially for Tank game. We are from Department of Computer Science and Engineering, University of Moratuwa. We are taking programming challenge II module as we are currently in semester 4. As the semester project of that subject, we were asked to create a client application that has
1. A seperate User Interface for the client side.
2. An Artificial Intelligence for play the game.
3. Connecting and synchronizing facilities which enable to access the game from a remote place
So we are among the teams that are creating that client application. At the end of the semester, a Final Competition between teams are going to happen. So we are in a mission to create the best AI that can beat any kind of opponent.
